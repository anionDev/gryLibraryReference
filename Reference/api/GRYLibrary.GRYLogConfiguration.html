DebugBreakMode

If true then System.Diagnostics.Debugger.Break will be executed every time a logentry will be created whose loglevel is contained in DebugBreakLevel. This feature is for debugging-purposes only.

Declaration
public bool DebugBreakMode { get; set; }

Remarks

Caution: Do not enable this function in productive-mode unless you know exactly what you are doing! Applications typically crash if System.Diagnostics.Debugger.Break will be executed and no debugger is attached. For that reason this function is disabled by default.
