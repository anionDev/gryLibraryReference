Enum Verbosity

This enum contains the verbosity-level for ExternalProgramExecutor.

Namespace: GRYLibrary.Core.Miscellaneous
Assembly: GRYLibrary.dll
Syntax
public enum Verbosity

Fields

Name Description
Full

Logs StdOut and StdErr of the executed program in realtime.
Normal

If the exitcode of the executed program is not 0 then the StdErr will be logged. This is the default-value for Verbosity.
Quiet

No output will be logged.
Verbose

Same as Full but with some more information added by ExternalProgramExecutor.
